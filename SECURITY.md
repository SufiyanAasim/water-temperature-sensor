# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 1.x (Ion) | ✅ |
| 0.x (Atom, pre-release) | ❌ |

## Reporting a Vulnerability

This is an offline Arduino sketch with no network surface, but if you find a
safety-relevant issue (e.g. a fault path that could suppress the
high-temperature alarm), please report it privately rather than opening a
public issue:

- Email: **sufiyanaasim@outlook.com**
- Include the sketch version, board, and steps to reproduce

You'll receive an acknowledgement within a few days. Please allow time for a
fix before public disclosure.
